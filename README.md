# Autotrack
AutoTrack is a smart automobile management and reminder system built with Java Spring Boot, Bootstrap, and Firebase. It helps vehicle owners manage insurance, PUC, and service records while providing automated email reminders for important deadlines. The system ensures better vehicle maintenance, convenience, and secure digital record management.


AutoTrack is a smart automobile management and reminder system built using Java Spring Boot, Bootstrap, Firebase, and MySQL. The project helps vehicle owners efficiently manage insurance renewals, PUC expiry dates, vehicle servicing schedules, and maintenance records.

The system provides secure user authentication, vehicle management, automated email reminders, and digital record storage. AutoTrack reduces the risk of missing important vehicle-related deadlines and improves overall vehicle maintenance through timely notifications and a user-friendly interface.

Key Features:
• User Registration and Login
• Vehicle Information Management
• Insurance & PUC Expiry Tracking
• Service Schedule Reminders
• Automated Email Notifications
• Secure Data Storage with Firebase
• Responsive UI using Bootstrap

This project demonstrates the implementation of modern web development technologies and best practices for building a complete automobile management solution.
